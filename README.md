### 1. Clone the repository
### 2. Database Setup
1. Open the Hospital-backend folder.
2. Locate `appsettings.json` and update the connection string with your local SQL Server

```bash dotnet run ```

### 3. Running the Backend (.NET)
``` cd Hospital-backend ```

``` dotnet run ```

### 4. Running the Frontend (Angular)
